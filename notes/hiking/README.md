# Hiking

Bring:

* Water
* Snacks
* Plastic Bag
  * Dirty Shoes
  * Pick up trash people have left behind
* Spare slippers/shoes
* Spare clothes

Apple Watch:

* Choose "Hiking"

Gotchas:

* 1km can be very different depending on the terrain
* If its rainy season it can be more dangerous
* Muddy and slippery
