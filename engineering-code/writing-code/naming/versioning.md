# Versioning

### String

```text
$product-$stage-$name
```

### Semantic Versioning \(Numeric\)

[https://semver.org/](https://semver.org/)

```bash
$major.$minor.$patch
```

#### Calendar Versioning \(CalVer\)

[https://calver.org/](https://calver.org/)

```bash
YYYY.MM.DD
```

Mix:

```bash
$major.$minor.YYYYMMDD.$patch
```
