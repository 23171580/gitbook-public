* Maintain composure
* Slow down
* Control your excitement.
* Carefully read the full question

## Technical

### Data structures:

Consider using one of:

* Array
* Linked Lists / Doubly Linked Lists
* Maps / Hashes
* Sets
* Binary Tree

### Algorithms:

* Binary Search
* Recursive
